Help: @heading

When a channel is in dev mode you can use versions of skills that are in active development and are not yet deployed for everyone.

You can enable dev mode for a channel by typing:

@@@{config.botName} enable dev mode

To turn off dev mode for that channel, you can type:

@@@{config.botName} disable dev mode
